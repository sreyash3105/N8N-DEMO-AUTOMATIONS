## 📥 Using These Workflows

All workflows are importable directly into any n8n instance:

1. Open n8n → **Workflows** → **Import from file**
2. Select a `.json` file
3. Re-link credentials when prompted (n8n will highlight what's needed)
4. Activate and run

> **Note:** Some workflows require third-party API keys (OpenAI, Apify, etc.). Each workflow's sticky notes describe what's needed to get it running.
